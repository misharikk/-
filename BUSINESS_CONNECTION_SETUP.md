# Настройка Business Connection для Premium аккаунта

## Зачем это нужно?

Нативные Telegram Checklists доступны только для Premium аккаунтов. Чтобы ваш бот мог создавать нативные чеклисты для обычных пользователей, нужно подключить Premium аккаунт через Business Connection.

## Пошаговая инструкция

### Шаг 1: Создайте Premium аккаунт

1. Откройте Telegram на новом номере телефона
2. Зарегистрируйте новый аккаунт
3. Оформите Telegram Premium подписку

### Шаг 2: Настройте Business Connection

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/mybots`
3. Выберите вашего бота (или создайте нового через `/newbot`)
4. Выберите пункт **"Business Connection"**
5. Нажмите **"Connect to Business"**
6. Выберите ваш Premium аккаунт
7. Подтвердите подключение

### Шаг 3: Получите business_connection_id

После подключения BotFather покажет вам `business_connection_id` - это строка вида:
```
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

Скопируйте этот ID.

### Шаг 4: Настройте бота

Отправьте вашему боту команду:
```
/set_business_connection <ваш_business_connection_id>
```

Например:
```
/set_business_connection 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

### Шаг 5: Готово!

После настройки все новые чеклисты будут создаваться в нативном формате Telegram, доступном для Premium аккаунтов.

## Важные замечания

- **Business Connection ID хранится в памяти** - при перезапуске бота нужно будет настроить заново (в будущих версиях можно сохранить в файл)
- **Только новые чеклисты** будут в нативном формате - старые останутся текстовыми
- **Premium аккаунт должен оставаться активным** - если подписка закончится, чеклисты перестанут работать

## Проверка настройки

Отправьте боту команду:
```
/set_business_connection
```
(без аргументов)

Бот покажет, настроен ли Business Connection.
