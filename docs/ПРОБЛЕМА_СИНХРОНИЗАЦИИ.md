# Проблема: Неправильная синхронизация выполненных задач

## Симптомы:
1. На скриншотах видно, что ВСЕ задачи в теговых чеклистах выполнены
2. В отчёте показывается только часть выполненных задач
3. На следующий день переносятся выполненные задачи

## Из логов:
- Теговые задачи частично синхронизируются
- В отчёте показывается только "1 теговых выполненных задач" вместо всех
- При переносе на следующий день переносятся задачи, которые должны были быть выполнены

## Корень проблемы:
События `checklist_tasks_done` приходят, но:
1. Не все задачи правильно определяются
2. Состояние может не сохраняться после обновления
3. При закрытии дня используется неправильное состояние

## Что нужно исправить:
1. Убедиться, что все события обрабатываются правильно
2. Проверить сохранение состояния после обновления
3. Убедиться, что при закрытии дня используются актуальные данные
